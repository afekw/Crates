# Crates
Crates is a skript that lets you make crates in game with a lot of features

## <ins>Commands<ins>:
- **/crates create** to add a crate.
- **/crates edit** to edit a crate.
- **/crates givekey** to give a key.
- **/crates givecrate** to give a crate block.
